ABOUT LEXACART

LEXACART is an online ecommerce project developed and designed by VISHWAS DESHPANDEY

Hi, I am Vishwas Deshpandey, and i am a MERN stack web developer and in this project the tech stack which is used is html css and javascript along with this the project uses localStorage which is used to store the cart products in a real time

In this Project all the Products Are not hard coded instead all the data is fetching dynamically using JavaScript, along with this the product detail of the product is also fetched dynamically in real time.

In this Project the Cart Functionallity is also diffrently developed from other projects here we are adding the cart items in this diffent ways:

  • If The Product not exist already in the cart then it will add a new product
  • If the Product is already Exist in the Cart and the size selected is same stored in the cart then the quantiy will increase
  • If the product is already added but size is diffrent then the new product is added in the cart
  • We can also change the qunatity of the product in the cart or also when adding the item in the cart it takes the real time quantiy and add in the cart
